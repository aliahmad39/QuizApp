# QuizApp
An android quiz app for taking and creating mcq's based quizzes and also added a spinner for entertainment implement in firebase.


# Functionality
- Register User
- Login with Email/Password
- Select Course
- Attempt Quiz
- Retake Quiz
- Display Result
- SpinWheel
- Earn Coins
- Withdraw Coins
- Update Profile
